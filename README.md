# remove-trash

Removes some trash files from the system, like `Thumbs.db`, `.DS_Store` and
`build` folders.

## It deletes

- `*.bak`
- `*.slxc`
- `.DS_Store`
- `.cache`
- `.gradle`
- `.mypy_cache`
- `.sass-cache`
- `.textpadtmp`
- `Thumbs.db`
- `__pycache__`
- `_build`
- `build`
- `slprj`
- `zig-cache`
- `zig-out`
- `~*`

## It ignores

- `.git`
- `.steam`
- `.var`
- `Steam`
- `containers`
